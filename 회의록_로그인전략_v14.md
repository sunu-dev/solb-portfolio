# SOLB 로그인 전략 — 전문가 패널 회의록 v14

**일시:** 2026년 3월 26일
**안건:** Google/Kakao 로그인 이메일 처리 전략

---

## 현재 문제

| 제공자 | 이메일 제공 | 상태 |
|--------|-----------|------|
| **Google** | ✅ 항상 제공 | 정상 동작 |
| **Kakao** | ❌ 이메일 못 받음 (비즈앱 전환 필요) | KOE205 에러 |

**KOE205 원인:** Supabase가 카카오에 `account_email` scope를 요청 → 카카오 동의항목에 이메일 권한 없음 → 거부

---

## 전문가 의견

### 백엔드 전문가

> **Google과 Kakao는 이메일 정책이 다릅니다:**
>
> | | Google | Kakao |
> |---|--------|-------|
> | 이메일 제공 | 항상 (기본) | **비즈 앱만** (사업자등록증 필요) |
> | 개인 개발자 | 이메일 받을 수 있음 | **이메일 못 받음** |
> | 사용자 식별 | 이메일로 | 카카오 고유 ID로 |
>
> **핵심: 카카오는 이메일 없이도 로그인이 가능합니다.**
> 카카오 고유 ID + 닉네임만으로 사용자를 식별할 수 있어요.

### PM

> **3가지 방안:**
>
> #### A안: 카카오에서 이메일 scope 제거 (코드 수정)
> - Supabase가 카카오에 이메일을 요청하지 않도록 scope 지정
> - 카카오 사용자는 이메일 없이 닉네임+ID로 가입
> - Google 사용자는 이메일 정상 수신
> - **장점:** 즉시 해결, 비용 없음
> - **단점:** 카카오 사용자 이메일 없음
>
> #### B안: 카카오 비즈 앱 전환
> - 사업자등록증 제출 → 비즈 앱 전환 → 이메일 동의항목 활성화
> - **장점:** 카카오에서도 이메일 수신
> - **단점:** 사업자등록증 필요, 심사 1~3일
>
> #### C안: 카카오 로그인 후 이메일 직접 입력 받기
> - 카카오 닉네임으로 로그인 → "이메일을 입력해주세요" 팝업
> - **장점:** 이메일 확보 가능
> - **단점:** 사용자 번거로움, 이탈 가능

### UX 전문가

> **A안을 추천합니다.**
>
> 이유:
> 1. 우리 서비스에서 이메일이 꼭 필요한가? → **아직 아님**
> 2. 이메일이 필요한 시점: 알림 발송, 비밀번호 찾기 → Phase 3
> 3. 지금은 **사용자가 로그인만 되면** 충분
> 4. 카카오 사용자는 닉네임으로 표시하면 됨
> 5. 나중에 비즈 앱 전환하면 이메일도 받을 수 있음
>
> **Google:** 이메일 자동 수신 → 그대로 사용
> **Kakao:** 닉네임 + 프로필사진만 → 충분

## 크로스 리뷰

### 백엔드 → UX
> 카카오 사용자가 나중에 이메일이 필요해지면, 프로필 설정에서 직접 입력하게 하면 됩니다. 지금 강제할 필요 없어요.

### PM → 백엔드
> 동일한 사람이 Google과 Kakao 둘 다 로그인하면 계정이 2개 생기지 않나요?

### 백엔드 → PM
> 네, 별도 계정이 됩니다. 계정 연동(Account Linking)은 Phase 3에서 처리하면 됩니다. 지금은 각각 독립 계정으로 가도 됩니다.

### 주식 초보 → 전체
> 저는 카카오로 로그인할 건데, 이메일 입력하라고 하면 귀찮아서 안 할 것 같아요. 그냥 카카오톡 닉네임으로 되면 좋겠어요.

## 합의: A안 (카카오 scope에서 이메일 제거)

> **구현:**
> ```typescript
> // useAuth.ts — 카카오 로그인 시 이메일 scope 제외
> const signInWithKakao = async () => {
>   await supabase.auth.signInWithOAuth({
>     provider: 'kakao',
>     options: {
>       redirectTo: window.location.origin,
>       scopes: 'profile_nickname profile_image',  // account_email 제외
>     },
>   });
> };
> ```
>
> **Google은 변경 없음** (이메일 자동 수신)
>
> **Supabase 설정:**
> - Kakao: "Allow users without an email" → ON (이미 함)
>
> **사용자 표시:**
> - Google: 이메일 + 이름 + 프로필 사진
> - Kakao: 닉네임 + 프로필 사진 (이메일 없음)
