# SOLB 총 금액 달러/원화 병기 전략 — 전문가 패널 회의록 v11

**일시:** 2026년 3월 25일
**안건:** 히어로 영역의 "총 금액"을 달러/원화로 어떻게 표시할 것인가

---

## 현재 상태

```
내 수익
-₩296.5만원        ← 원화만 크게
(-$197.60)         ← 달러는 작게, 회색
(-3.30%)

총 평가 ₩8,689만원 ($5,793)  │  총 투자 ₩8,986만원 ($5,990)  │  종목 4개
```

---

## 전문가 의견

### UX 전문가

> **3가지 안을 비교합니다:**
>
> #### A안: 현재 — 원화 메인, 달러 보조
> ```
> -₩296.5만원
> (-$197.60)
> ```
> 장점: 원화가 명확히 메인
> 단점: 달러가 너무 작아서 안 보일 수 있음
>
> #### B안: 나란히 병기
> ```
> -₩296.5만원 / -$197.60
> ```
> 장점: 한 줄에 둘 다 보임
> 단점: 숫자가 길어서 모바일에서 줄바꿈
>
> #### C안: 탭 전환 (토스 스타일)
> ```
> [₩ 원화]  [$ 달러]      ← 토글
>
> -₩296.5만원              ← 원화 모드
> (-3.30%)
>
> 또는
>
> -$197.60                 ← 달러 모드
> (-3.30%)
> ```
> 장점: 깔끔, 모바일 친화적
> 단점: 동시에 못 봄

### 주식 초보 A

> 저는 **C안**이 좋아요. 토스에서도 $ ₩ 전환 버튼이 있잖아요. 평소엔 원화로 보다가 궁금하면 달러로 전환하는 게 직관적이에요.

### 주식 초보 B

> 저도 C안이요. 숫자가 너무 많으면 헷갈려요. 한 가지만 크게 보여주는 게 좋아요.

### UI 전문가

> **C안 + 현재 방식 혼합이 최적입니다.**
>
> ```
> [$]  [₩]                    ← 작은 토글 (토스 사이드바와 동일)
>
> -₩296.5만원                 ← 선택한 통화로 메인 표시
> (-3.30%)
>
> 총 평가 ₩8,689만원  │  총 투자 ₩8,986만원  │  종목 4개
> ```
>
> 토글 위치: 히어로 오른쪽 상단
> 토글 크기: 작게, 24px 높이, 둥근 pill
> 선택된 통화: 진한 배경, 반대쪽: 연한
>
> **달러 모드일 때:**
> ```
> [$]  [₩]
>
> -$197.60
> (-3.30%)
>
> 총 평가 $5,793  │  총 투자 $5,990  │  종목 4개
> ```
>
> **종목 리스트도 연동:**
> - 원화 모드: -₩159.5만 (현재)
> - 달러 모드: -$106.30

### PM

> **토스증권이 정확히 이 방식입니다.**
> 토스 사이드바에 `$ ₩` 토글이 있고, 전환하면 모든 금액이 바뀝니다.
>
> 우리도 이걸 적용하면:
> 1. 히어로 금액이 깔끔해짐 (숫자 하나만)
> 2. 초보자가 헷갈리지 않음
> 3. 토스 사용자에게 익숙한 UX

## 합의

### 전원 합의: **C안 ($ ₩ 토글)**

> | 항목 | 내용 |
> |------|------|
> | **토글 위치** | 히어로 영역 오른쪽 상단 |
> | **토글 디자인** | 작은 pill: `[$]` `[₩]`, 활성=진한 배경, 비활성=회색 |
> | **기본값** | ₩ 원화 |
> | **영향 범위** | 히어로 수익, stats, 종목 리스트 내 수익, 분석 패널 |
> | **한국 주식** | 항상 원화 (토글 무관) |
> | **저장** | localStorage에 선호 통화 저장 |
>
> **기존 괄호 달러 표시 제거** → 토글로 전환하는 방식이 더 깔끔
