# SOLB PORTFOLIO 전략 재정립 — 전문가 패널 회의록 v5

**일시:** 2026년 3월 24일
**안건:** 서비스 목표 재확인 + 핵심 가치에 맞는 기능 재설계
**핵심 목표:** "내 수익이 얼마인지, 목표까지 얼마나 남았는지, 관련 지표와 뉴스를 한눈에 보여주는 초보자 투자 도우미"

---

## 서비스 목표 재확인

### PM

> **핵심 질문 3개에 답하는 서비스:**
>
> 1. **"나 지금 얼마 벌고 있어?"** → 내 수익/손실을 가장 먼저, 가장 크게
> 2. **"목표까지 얼마나 남았어?"** → 목표 달성도 시각화
> 3. **"지금 상황이 어때?"** → 관련 지표 + 뉴스 + 차트 모양 해석
>
> 이 3개에 답하지 못하는 기능은 **후순위**입니다.
> 매크로 지표 6개 나열, 기술적 지표 3개 카드 — 이런 건 초보자에게 부담만 됩니다.
> **"내 돈이 어떻게 되고 있는가"**에 집중해야 합니다.

## 1부: "나 지금 얼마 벌고 있어?" — 수익 표시 전략

### UX 전문가

> **현재 문제:** 수익이 테이블 맨 오른쪽 작은 글씨(+1.9%)로 표시. 초보자가 찾기 어려움.
>
> **원칙: 수익/손실이 가장 먼저 눈에 들어와야 합니다.**
>
> #### 메인 대시보드 구성 (우선순위 순서)
>
> ```
> ┌─────────────────────────────────────────────┐
> │                                             │
> │  내 투자 수익                                │  ← 1번: 전체 수익
> │                                             │
> │  +₩18만원                                   │  ← 원화로! (달러 아님)
> │  +1.17%                                     │
> │                                             │
> │  총 평가  ₩1,598만원                         │
> │  총 투자  ₩1,580만원                         │
> │                                             │
> │  ──────────────────────────────────────────  │
> │                                             │
> │  종목별 수익                                  │  ← 2번: 종목별 수익
> │                                             │
> │  [M] 마이크론    +₩1.8만  +1.17%  ██████░░  │  ← 목표 달성 바
> │  [M] 마이크로..  -₩6.8만  -2.34%  ████░░░░  │
> │  [B] BEX        -₩2.1만  -12.7%  █░░░░░░░  │  ← 빨간 바 (하락)
> │                                             │
> │  ──────────────────────────────────────────  │
> │                                             │
> │  📰 내 종목 뉴스                              │  ← 3번: 관련 뉴스
> │                                             │
> │  마이크론 실적 발표 D-7...                    │
> │  MSFT 클라우드 매출 성장 지속...               │
> │                                             │
> └─────────────────────────────────────────────┘
> ```
>
> **핵심 변경:**
> - 첫 화면에 **원화 수익**이 가장 크게
> - 종목별로 **수익 + 목표 달성 바**가 바로 보임
> - 바로 아래에 **내 종목 관련 뉴스**
> - 차트/지표/이벤트는 **종목 탭 시** 상세에서

### 주식 초보 A

> 이거예요! 저는 매일 아침 "오늘 내가 플러스야 마이너스야"만 알고 싶거든요. 지금은 테이블에서 숫자를 찾아야 하는데, 이렇게 크게 보여주면 바로 알 수 있어요.

## 2부: "목표까지 얼마나 남았어?" — 목표 추적

### UI 전문가

> **목표 달성을 시각적으로 보여주는 3가지 방법:**
>
> #### 방법 1: 프로그레스 바 (추천)
> ```
> 마이크론  목표 +20%
> ████████████░░░░░░░░░░░░░  현재 +1.17%  (5.9% 달성)
> ```
> - 직관적, 게이미피케이션 효과
> - 초보자가 바로 이해 가능
> - 목표 도달 시 축하 효과
>
> #### 방법 2: 원형 차트
> ```
>    ╭──╮
>   ╱ 5.9%╲
>  │  달성  │
>   ╲    ╱
>    ╰──╯
> ```
> - 공간 효율적
> - 모바일에 적합
>
> #### 방법 3: 숫자 + 색상
> ```
> 목표 $502  │  현재 $422.90  │  남은 거리 +18.7%
> ```
> - 가장 간단
> - 정보 밀도 높음
>
> **추천: 리스트에는 프로그레스 바, 상세에는 숫자+프로그레스 바 조합**

### 주식 초보 B

> 프로그레스 바가 좋아요! 게임에서 레벨업 하는 느낌이에요. "아 5%밖에 안 됐네" → "20%까지 가자!" 이런 동기부여가 되거든요. 목표 달성하면 뭔가 축하 이펙트도 있으면 좋겠어요!

## 3부: "지금 상황이 어때?" — 차트 모양 인식

### 주식 고수

> **초보자에게 차트를 "읽어주는" 것이 우리의 핵심 차별점입니다.**
>
> 현재 구현된 것:
> - SMA(5/20/60일), RSI, 패턴 인식, 골든크로스/데드크로스
>
> **부족한 것: "그래서 지금 차트가 어떤 모양이야?"를 한 문장으로**
>
> #### 차트 모양 요약 카드 (제안)
>
> ```
> ┌──────────────────────────────────────┐
> │  📊 MU 차트 요약                     │
> │                                      │
> │  ╭────╮                              │
> │  │ 📉  │  하락 추세에서 바닥 다지는 중  │
> │  ╰────╯                              │
> │                                      │
> │  "최근 30일간 내려갔지만, 지금 가격이   │
> │   더 이상 안 떨어지는 바닥 구간이에요.   │
> │   과거에 이런 모양에서 70% 확률로       │
> │   반등이 나왔어요."                    │
> │                                      │
> │  🟡 관망 추천  │  RSI 32 (과매도)      │
> │                                      │
> │  쉬운 설명 보기 ▸                     │
> └──────────────────────────────────────┘
> ```
>
> **초보자가 이해할 수 있는 차트 모양 분류:**
>
> | 차트 모양 | 아이콘 | 초보자 설명 | 신호 |
> |----------|-------|-----------|------|
> | 상승 추세 | 📈 | "꾸준히 올라가고 있어요" | 🟢 |
> | 하락 추세 | 📉 | "계속 내려가고 있어요" | 🔴 |
> | 바닥 다지기 | 🔄 | "더 이상 안 떨어지는 구간이에요" | 🟡 |
> | 고점 부근 | ⚠️ | "많이 올라서 조심할 구간이에요" | 🟡 |
> | 횡보 (박스권) | ➡️ | "일정 범위에서 왔다갔다 해요" | 🟡 |
> | 급락 후 반등 | ⬆️ | "많이 떨어졌다가 다시 올라오는 중이에요" | 🟢 |
> | 급등 후 조정 | ⬇️ | "많이 올랐다가 쉬어가는 중이에요" | 🟡 |
>
> **"쉬운 설명 보기"를 탭하면:**
> - "이동평균선이 뭔가요?" → 한 줄 설명
> - "RSI 32가 뭔 뜻이에요?" → "0~100 사이 숫자로, 30 이하면 너무 많이 떨어졌다는 뜻이에요"
> - "이런 모양에서 앞으로 어떻게 되나요?" → 과거 통계 기반 설명

### 애널리스트

> 차트 모양 분류는 기존 `detectPattern()`, `detectTrend()`, `calcRSI()` 함수를 조합하면 됩니다.
> 새로운 함수가 필요한 게 아니라, **기존 분석 결과를 초보자 언어로 번역하는 레이어**만 추가하면 됩니다.
>
> ```
> function getChartSummaryForBeginner(trend, pattern, rsi, cross) {
>   // 기존 분석 결과 → 초보자 한 문장
>   if (trend === 'strong_down' && rsi < 30)
>     return { icon: '🔄', title: '바닥 다지는 중', desc: '많이 떨어져서...' };
>   ...
> }
> ```

## 4부: 관련 뉴스와 지표를 보기 편하게

### UX 전문가

> **현재 문제:** 뉴스가 별도 탭에 있어서 "내 종목" → "뉴스" 왔다갔다 해야 함.
>
> **제안: 뉴스를 종목에 붙이기**
>
> ```
> 메인 화면
> ├── 내 수익 요약 (히어로)
> ├── 종목별 수익 리스트
> │   ├── 마이크론 +1.17%  [차트📊] [뉴스📰]  ← 아이콘 탭하면 해당 종목 뉴스/차트
> │   └── ...
> ├── 📰 내 종목 관련 뉴스 (최신 5개)  ← 메인에 바로 보이게
> │   ├── "마이크론 실적 발표 D-7" - 한국경제 - 2시간 전
> │   └── "MSFT 클라우드 매출 성장" - 조선비즈 - 3시간 전
> └── 📊 시장 한줄 요약
>     └── "S&P 500 -1.51%, 나스닥 -2.00% — 기술주 중심 하락"
> ```
>
> **별도 뉴스 탭도 유지**하되, 메인에서 **내 종목 관련 뉴스 3~5개**를 바로 보여줌.
> 이러면 메인 화면만으로 "수익 + 뉴스 + 시장 상황" 3가지를 한눈에 파악 가능.

>
> | 순서 | 영역 | 보여주는 것 |
> |------|------|-----------|

> | 2 | **목표 달성도** | 프로그레스 바 + 목표가 |
> | 3 | **차트 모양 요약** | "바닥 다지는 중" 한 문장 + 신호등 |

> | 5 | **캔들 차트** | SMA 오버레이 |
> | 6 | **관련 뉴스** | 해당 종목 뉴스 |
> | 7 | **기술 지표 상세** | RSI, 패턴 등 (접을 수 있게) |
>
> #### 사이드바 (빠른 확인)
>
> | 영역 | 보여주는 것 |
> |------|-----------|
> | SOLB AI | 가장 중요한 알림 한 줄 |
> | 관심 종목 | 가격 + 등락률 빠른 확인 |
>
> #### 뉴스 탭 (깊이 보기)
> - 전체/미장/국장/관심종목/인기
> - 뉴스 상세 보기

> #### 이벤트 탭 (학습용)
