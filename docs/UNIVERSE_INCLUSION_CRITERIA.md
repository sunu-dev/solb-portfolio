# Universe 편입 객관 기준 (Universe Inclusion Criteria)

> **시행일**: 2026-05-15 (베타 출시일과 동시)
> **버전**: v1.0
> **법적 위치**: 자본시장법 회피용 외부 공개 문서. 종목 추천이 아니라 **객관 기준 기반 큐레이션**임을 명시.

---

## 1. 목적

주비(이하 "서비스")는 검색·분석·알림 대상이 되는 종목을 **사전에 정의된 객관 기준**으로만 큐레이션합니다. 이 문서는 그 기준을 투명하게 공개해 다음을 명확히 합니다:

- 서비스는 **특정 종목을 추천하지 않습니다.**
- 편입 여부는 **사람의 주관이 아닌 객관적 수치 조건**으로만 결정됩니다.
- 사용자는 검색·즐겨찾기를 통해 **자신의 판단으로** 어떤 종목이든 등록할 수 있습니다.

자본시장법상 투자자문업·유사투자자문업에 해당하지 않으며, 어떠한 형태의 종목 매수·매도 권유도 하지 않습니다.

---

## 2. 편입 3중 AND 기준 (Universe ⊂ All Listings)

다음 **3가지 조건을 모두 만족**한 종목만 자동으로 'eligible' 상태로 승급되어 universe에 편입됩니다.

| # | 기준 | 조건 | 근거 |
|---|---|---|---|
| 1 | **시가총액** | $5,000,000,000 (USD 50억) 이상 | 유동성 확보, 펌프&덤프 대상 회피 |
| 2 | **상장 경과 기간** | **상장 후 만 12개월 이상** | IPO 직후 변동성·데이터 부족 회피 |
| 3 | **데이터 정상성** | 과거 30일 시세·기본 재무 데이터 모두 정상 수신 | 데이터 누락·오류 종목 자동 제외 |

세 조건 중 **하나라도 충족하지 못하면** universe에 편입되지 않으며, 검색 결과에는 노출되지만 "📅 신규" 배지 또는 "데이터 제한" 안내가 부착됩니다.

조건은 매일 1회 자동 재평가되며, 변동 시 상태가 자동 갱신됩니다. (예: 시총이 $5B 아래로 떨어지면 status가 'watch'로 강등)

---

## 3. 상태 머신 (State Machine)

종목은 `stock_listings` 테이블에서 다음 5개 상태 중 하나를 가집니다:

```
detected (감지) → watch (관찰) → eligible (편입 후보) → universe (편입) → delisted (상폐)
                                                       ↓
                                                  rejected (배제)
```

| 상태 | 의미 | 사용자 노출 |
|---|---|---|
| `watch` | 신규 감지 + 검수 전 | 검색 결과 + "📅 신규" 배지 |
| `eligible` | 3중 AND 충족, 한국어명 미입력 | 검색 결과만 |
| `universe` | 3중 AND 충족 + 한국어명 입력 완료 | AI 분석·알림·추천 대상 모두 |
| `rejected` | 운영자 수동 배제 (예: 페니주식, 테마주) | 검색 노출 X |
| `delisted` | Finnhub fetch에서 30일 연속 부재 | 검색 노출 X (보유자에 알림만) |

---

## 4. 검수 절차

자동 승급된 'eligible' 종목이 'universe'로 진입하려면 **운영자의 수동 검수 1회**가 필요합니다. 검수 항목:

1. **한국어 종목명 입력** (필수). 자동 번역(Gemini) 후 운영자가 검토·확정.
2. **간단 설명** (선택). "AI 반도체 대장주" 등 1줄.
3. **거래소 분류** 정확성 확인.
4. **이상 거래 의혹** 점검 (펌프&덤프, 작전주 의심).

검수 통과 시 'universe'로 승급되어 다음 cron부터 AI 분석·알림 대상에 포함됩니다.

## 5. 한국 종목 우회 (KRX 직접 등록 임시 정책)

Finnhub 무료 플랜은 KRX(KS·KQ) 데이터를 제공하지 않아, 한국 종목은 다음 두 경로로만 stock_listings에 등록됩니다:

1. **사용자 검색 자동 등록**: 사용자가 `.KS` 또는 `.KQ` suffix로 검색하면 자동으로 `status='watch'`로 insert. 시간 누적으로 자연스러운 universe 형성.
2. **운영자 수동 추가**: `/admin/listings` 의 "+ 수동 추가" 버튼. 거래소·심볼·한국어명 직접 입력.

**향후 계획**: KRX OpenAPI 또는 OpenDART 자동 fetch cron 도입. 월 10건 이하 신규 IPO라 임시 정책으로 운영 가능.

## 6. 사용자 입력 종목 (Out-of-Universe)

사용자가 universe 외 종목을 검색·등록하는 것은 **언제나 자유**입니다. 이 경우:

- 시세·차트·기본 분석은 정상 제공
- AI 촉(추천 엔진)에서는 제외 (객관 기준 미달)
- "데이터 제한" 안내 부착
- 알림은 가격 변동만 (기술 지표 알림은 제한)

서비스는 사용자의 자율적 선택을 존중하며, **편입 기준 미달 종목에 대한 책임은 전적으로 사용자에게 있습니다.**
