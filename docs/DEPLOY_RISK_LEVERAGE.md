# 배포 위험 검토 및 결정 기록 — 레버리지 중간 옵션 + AI 방향 0

> **목적**: 변호사 정식 검토(§8) 전에 production 배포를 결정할 경우, **무엇이 법적으로 미해결인지**를 명확히 기록하고 파운더가 숙고된 결정을 내리기 위한 문서. 이 문서 자체가 "고의·중과실 아님 / 선의·주의의무 이행"의 증거가 된다.
> **작성**: 2026-05-30 · **갱신**: 2026-06-01 (AI 적대적 검토 기반 — 실제 법률 자문 아님)
> **현행 production**: 레버리지 전면 차단 + 약관 v3 (가장 보수적 상태)
> **배포 시 변경**: 레버리지 = 검색 노출 + 보유 관리 + **AI 분석 거부**(보유 해설 아님) + AI 전 종목 매매 방향 0 + 약관 v4
>
> **🔄 2026-06-01 설계 변경**: 레버리지 AI 분석을 '보유 위험 해설' → **'분석 거부 + 일반 종목 유도'**로 전환(디자인 패널). 이로써 **리스크 2(적합성 의무 상품 해설)가 대부분 해소** — 해설을 안 하고 거부하므로. 아래 리스크 2 갱신 참조.

## 0. 한 줄 요약

**코드는 견고합니다(3라운드 적대적 검증, 방향 0, 게이트, deny-list). 막는 건 코드가 아니라 "법적 클리어런스 3건"이고, 이건 AI가 사인할 수 없는 실제 변호사 영역입니다.** 지금 배포 = 아래 3개 미해결 리스크를 파운더가 감수하는 결정.

## 1. 미해결 법적 리스크 3건

### 🔴 리스크 1 — 약관 v4 실제 변호사 미검토
- **무엇**: 약관 v4(제2조 매매없음·관리중심 / 제5조 취급범위 / 제7조 면책)는 **AI(나)가 작성한 초안**으로, 자본시장법 전문 변호사 검토를 거치지 않음.
- **법조/위험**: 면책 조항(제7조)에 흠결이 있으면, 손실 분쟁에서 면책이 무효화되거나 다투어질 수 있음.
- **가능성·심각도**: 가능성 中(텍스트는 보수적·정책 정합이나 미검증) / 심각도 中(면책 방어력).
- **완화됨**: 동의 버전 정합(v4 일치), 14세→성인+위험 게이트 동의 로깅, 정책과 문구 정합.

### 🟢 리스크 2 — 16종 ETF '적합성 의무 잠탈' (2026-06-01 거부 설계로 대부분 해소)
- **무엇(원래)**: 2026-05-27 KRX 단일종목 레버리지 ETF 16종은 **금감원·KRX가 사전교육·예탁금·적합성 확인을 부과**한 상품. 이걸 (보유 시) 위험 **해설**하는 것이 적합성 절차 잠탈로 해석될 여지였음.
- **🔄 해소**: 2026-06-01 설계 변경으로 레버리지는 **AI 분석/해설을 전혀 안 하고 거부**(고위험·적합성 사유 안내 + 일반 종목 유도). 남는 건 '검색 노출'뿐인데 이는 **"검색 표시만은 합법"** 레인. 즉 '적합성 상품을 분석/해설'하는 행위가 사라져 잠탈 우려가 거의 없어짐.
- **잔존(약함)**: 검색 노출 자체에 대한 규제당국 스탠스 미확립(신상장). 단 노출+거부는 노출+해설보다 훨씬 안전. 만 19세+위험 동의 게이트·deny-list(분석·촉·universe 차단) 유지.

### 🔴 리스크 3 — 사전 법률의견서 부재
- **무엇**: 의견서 §5가 명시 — "정식 GO와 약관 v4 검토 전 코드 배포 금지." 사전 법률의견서 없이 배포.
- **법조/위험**: 분쟁·금감원 조사 시 "선의·주의의무 이행" 입증 자료 부족 → 다른 리스크 증폭.
- **가능성·심각도**: 분쟁 발생 시에만 작동 / 심각도 中(다른 리스크 증폭).
- **완화됨**: **이 문서(숙고된 결정 기록) + 3라운드 적대적 검증 로그 + 약관 면책**이 부분적으로 선의 입증을 보완.

## 2. 이미 완화된 것 (배포해도 깔린 안전망)

- **3라운드 적대적 검증** loop-until-clean — reachable §6 누수 0 확인.
- **AI 매매 방향 0** — 모든 종목, 프롬프트 대원칙 + 결정론적 서버 백스톱(`enforceLeverageReport`) + FORBIDDEN_PHRASES.
- **단일종목 레버리지**: 신규 추천·AI촉·universe·매수 유인 차단(deny-list 18종) + 보유 시 위험 해설만 + 고위험 라벨 + 위험 동의 게이트.
- **무료 유지**(§101 신고 면제) · 객관 기준 큐레이션 · 환각 통계·확률 금지 · 동의 시점 DB 로깅.
