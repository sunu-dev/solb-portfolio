## 1. 한 줄 정의

**주비(joobi)** = 한국 개인투자자가 **본인이 직접 입력(또는 증권사 화면 캡처 OCR)한 주식 보유 정보**를 관리·확인하고, 그에 대한 **방향성 없는 정보·해설**을 받는 모바일 웹앱(PWA). **무료 베타.**

**주비는 증권사·거래소·자문사가 아닙니다.** 매매 체결·중개·주문, 종목 추천, 투자 자문(매수/매도 방향·목표가·매수 시점)을 **하지 않습니다.**

## 4. 수익화 구조 (§101 정합 핵심)
- **무료 (영구)**: AI 분석·AI 촉·digest·알림·뉴스·이벤트 = **모든 투자 정보**. → '투자조언에 대가 없음' = §101 면제 전제 보존.
- **PRO ₩4,900/월 (예정, 미구현)**: 투자판단이 **아닌 도구**만 잠금 — 포트폴리오 시각화 심화, **세무 신고용 정리 리포트**, 데이터 백업 등. AI 한도 상향은 묶음에 포함하되 '투자조언 자체에 과금'으로 해석되지 않게 product·약관 분리 표기.
- 광고·매매수수료·일임보수 **없음.**

## 6. 규제 박스 대조 (왜 주비가 어디에도 정확히 안 들어가나)
| 사업자 | 인가/신고 | 주비와 차이 |
|---|---|---|
| 증권사 | 투자중개 인가 + 적합성 절차 직접 운영 | 주비는 중개·체결 안 함 |
| 정보제공앱(유사투자자문) | §101 신고(유료·불특정다수 투자조언) | 주비는 투자조언 **무료** + **방향0**(조언 아님) |
| 로보어드바이저 | 투자자문/일임 인가 | 주비는 방향·일임 안 함 |
| 세무 신고대행 | 세무사 자격 | 주비는 계산·정리만, 신고 셀프 |
→ 주비는 **"초개인화 정보 제공 + 본인 데이터 관리 도구"**로, 위 면허 박스 밖 레인을 의도. **이 레인 적정성 검증이 자문의 핵심.**
