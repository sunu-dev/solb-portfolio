## 1부: "진짜 AI" vs "규칙 기반" — 어디까지 할 것인가?

### PM

> **3단계로 나눕니다:**
>
> #### Level 1: 스마트 규칙 (지금 가능, $0)
> - 기존 `technical.ts` 함수들을 조합해서 더 다양한 상황 감지
> - 텍스트 템플릿을 풍부하게 확장
> - 30초마다 자동 갱신 (현재 auto-refresh에 연동)
>
> #### Level 2: Claude API 연동 (Phase 2, ~$10~30/월)
> - 차트 데이터 + 뉴스를 Claude에게 보내서 자연어 분석
> - 하루 1~2회 배치 분석, 결과 캐싱
> - 진짜 "AI"라고 부를 수 있는 수준
>
> #### Level 3: 실시간 AI + 푸시 알림 (Phase 3)
> - 목표가/손절가 도달 시 푸시 알림
> - 시장 급변 시 자동 분석 트리거
> - PWA Service Worker 또는 Supabase Realtime

### 전원 합의

> **지금은 Level 1을 제대로 구현하자.** Claude API 비용 없이도 충분히 유용한 알림을 만들 수 있다.

## 3부: 알림을 어떻게 "지속적으로" 제공할 것인가?

### UX 전문가

> #### 알림 표시 위치 3곳
>
> **① 사이드바 SOLB AI 영역 (현재)**
> - 가장 중요한 알림 2~3개만 표시
> - 30초마다 자동 갱신 (auto-refresh와 동기화)
> - 우선순위: 긴급 > 리스크 > 기회 > 인사이트
>
> **② 종목 행에 인라인 배지 (신규)**
> ```
> [M] 마이크론  $422  ▼-0.98%  +₩1.8만  ████  🟡 RSI 32
>                                              ↑ 종목별 알림 배지
> ```
> - 해당 종목에 활성 알림이 있으면 작은 배지로 표시
> - 탭하면 상세 설명
>
> **③ 토스트 알림 (신규)**
> ```
> ┌──────────────────────────────────┐
> │ 🎯 마이크론이 목표가에 근접!      │
> │ 현재 $422 / 목표 $430 (1.9% 남음) │
> │                        [확인]    │
> └──────────────────────────────────┘
> ```
> - 화면 상단에 3초간 표시 후 자동 사라짐
> - 중요한 상황 변화 시에만 (모든 갱신마다 뜨면 피로)
> - **한 번 표시된 알림은 같은 세션에서 반복 안 함**

### FE 전문가

> #### 구현 방식
>
> ```
> refreshAllData()
>   ├── 주가 데이터 fetch
>   ├── 캔들 데이터 fetch
>   ├── checkAllAlerts(stocks, macroData, rawCandles)  ← 신규
>   │   ├── 가격 기반 알림 체크 (1~10번)
>   │   ├── 기술 분석 기반 알림 체크 (11~18번)
>   │   └── 포트폴리오 전체 알림 체크 (19~20번)
>   ├── 결과 → alertsStore에 저장
>   └── UI 자동 갱신 (사이드바 + 배지 + 토스트)
> ```
>
> **Store에 alerts 상태 추가:**
> ```typescript
> alerts: Alert[]  // { id, symbol, type, message, severity, timestamp, dismissed }
> ```
>
> **토스트는 중복 방지:**
> ```typescript
> // 같은 세션에서 같은 alert ID는 다시 표시 안 함
> dismissedAlerts: Set<string>
> ```

## 4부: Level 2 — Claude API 연동 (Phase 2 예고)

### PM

> Level 1이 안정화되면 Claude API를 추가합니다:
>
> | 항목 | 내용 |
> |------|------|
> | 호출 시점 | 하루 1회 (장 마감 후) 또는 사용자 요청 시 |
> | 입력 | 최근 30일 캔들 + RSI/SMA/MACD + 관련 뉴스 3개 |
> | 출력 | 자연어 분석 리포트 (현재 상태 + 전망 + 주의사항) |
> | 비용 | ~$0.01~0.03/종목/분석 |
> | 저장 | Supabase DB에 캐싱 (같은 날 재요청 시 캐시 반환) |
> | 모델 | Claude Haiku (빠르고 저렴) 또는 Sonnet (깊은 분석) |
>
> **프롬프트 예시:**
> ```
> 한국인 주식 초보자를 위해 다음 종목을 분석해주세요.
> - 매수/매도 추천은 하지 마세요 (법적 이슈)
> - "~에요", "~해요" 체로 친근하게
> - 전문 용어는 괄호 안에 쉬운 설명 추가
>
> 종목: MU (마이크론)
> 최근 30일 종가: [...]
> RSI: 32
> SMA20: $435
> 현재가: $422
> 관련 뉴스: "마이크론 실적 발표 D-7"
> ```
