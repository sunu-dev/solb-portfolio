## 2부: 종목(티커) 관리 — 어떻게 할 것인가?

### UX 전문가

> **현재 문제점:**
> 1. 종목을 "단기/장기/관심"으로만 분류 — 초보자에겐 이 구분이 모호
> 2. 종목 추가 시 매수 정보를 바로 안 넣음 — 나중에 편집해야 함
> 3. 여러 번에 나눠 매수한 경우 (물타기) 관리 불가
>
> **제안하는 종목 관리 구조:**
>
> ```
> 내 종목
> ├── 투자 중 (실제 보유)
> │   ├── MU 마이크론
> │   │   ├── 1차 매수: 2026.2.15 / $430 / 10주
> │   │   ├── 2차 매수: 2026.3.05 / $410 / 15주
> │   │   └── 평균 단가: $418 / 총 25주 / 투자금 $10,450
> │   ├── MSFT 마이크로소프트
> │   │   └── 매수: 2026.3.10 / $380 / 20주
> │   └── ...
> ├── 관심 종목 (매수 전 관찰)
> │   ├── AMZN 아마존 — "$190 이하면 매수 예정"
> │   └── TSLA 테슬라 — "관망 중"
> └── 과거 종목 (매도 완료)
>     └── AAPL 애플 — 매수 $170 → 매도 $185 / 수익 +8.8%
> ```
>
> **핵심 변경:**
> - "단기/장기" 구분 폐지 → **"투자 중" / "관심 종목" / "과거 종목"**
> - "투자 중"에는 반드시 매수 정보 입력 (가격, 수량, 날짜)
> - **분할 매수(물타기) 지원** — 같은 종목에 여러 번 매수 기록
> - "관심 종목"에는 메모 + 목표 매수가
> - "과거 종목"은 매도 기록 + 실현 수익

### 주식 초보 A

> 저는 마이크론을 3번에 나눠서 샀는데, 지금 앱에서 평균 단가를 직접 계산해서 입력해야 해요. 너무 불편해요.
> "몇 월 몇 일에 얼마에 몇 주 샀다"를 기록하면 자동으로 평균 단가를 계산해주면 좋겠어요.

### 주식 초보 B

> "단기 트레이딩"이 뭔지 모르겠어요. 저는 그냥 "내가 산 주식"이랑 "사고 싶은 주식"만 구분되면 돼요.

## 3부: 매수한 주식을 어떻게 보여줄 것인가?

### UI 전문가

> **현재 문제:** 테이블에 종목명 + 가격 + 등락률만 보임. "내가 얼마를 투자했고, 얼마를 벌었는지"가 한눈에 안 들어옴.
>
> **제안하는 "투자 중" 종목 카드 디자인:**
>
> ```
> ┌────────────────────────────────────────────────────────────┐
> │                                                            │
> │  [M] 마이크론                                    MU   ▸    │
> │                                                            │
> │  현재가                 내 평균 단가                         │
> │  $422.90               $418.00                             │
> │                                                            │
> │  ┌──────────────────┐  ┌──────────────────┐               │
> │  │  오늘 등락        │  │  내 수익/손실     │               │
> │  │  ▼ -$4.16        │  │  +$122.50        │               │
> │  │  -0.98%          │  │  +1.17%          │               │
> │  │  (파랑=하락)      │  │  (빨강=수익)      │               │
> │  └──────────────────┘  └──────────────────┘               │
> │                                                            │
> │  25주 보유  │  투자금 $10,450  │  평가금 $10,572            │
> │                                                            │
> │  원화 환산: 약 1,580만원 (투자) → 약 1,598만원 (현재)       │
> │                                                            │
> │  목표 수익률 20% ███████░░░░░░░░░░░░ 5.9% 달성             │
> │                                                            │
> │  매수 이력                                                  │
> │  · 2026.2.15  $430 × 10주                                  │
> │  · 2026.3.05  $410 × 15주                                  │
> │                                                            │
> └────────────────────────────────────────────────────────────┘
> ```
>
> **핵심 포인트:**
> 1. **"오늘 등락"과 "내 수익/손실" 분리** — 초보자가 가장 헷갈리는 부분
> 2. **원화 환산** — 한국인에게 달러보다 원화가 체감 빠름
> 3. **목표 달성 프로그레스 바** — 게이미피케이션 효과
> 4. **매수 이력** — 언제 얼마에 샀는지 기록

### UX 전문가

> **화면 구성 제안 — 2가지 뷰:**
>
> #### 리스트 뷰 (기본 — 한눈에 전체 파악)
> ```
> [M] 마이크론     $422.90  ▼-0.98%   +$122 (+1.17%)   ▸
> [M] 마이크로소프트 $381.87  ▼-1.84%   -$456 (-2.34%)   ▸
> [A] 아마존 (관심) $205.37  ▼-1.62%   목표 $190         ▸
> ```
> - 왼쪽: 종목명
> - 가운데: 현재가 + 오늘 등락
> - 오른쪽: **내 수익/손실** (투자 중) 또는 **목표가** (관심)
> - 탭하면 상세 카드로 확장
>
> #### 카드 뷰 (상세 — 종목 탭 시)
> - 위에서 제안한 상세 카드
> - 분석 패널에서 보여줄 내용

### 주식 고수

> **"내 수익/손실"이 리스트에서 바로 보이는 게 핵심입니다.**
> 초보자가 가장 궁금한 건 "내가 돈을 벌고 있나 잃고 있나"예요.
> 오늘 등락률(-0.98%)보다 **내 수익률(+1.17%)**이 더 중요합니다.

## 5부: 크로스 리뷰

### 주식 초보 A → PM
> "투자 중/관심/과거" 분류가 직관적이에요. "단기 트레이딩"은 뭔지 몰랐는데 "투자 중"은 바로 이해돼요.

### 주식 초보 B → UI
> 카드에서 "오늘 등락"과 "내 수익/손실"이 나란히 보이는 게 좋아요. 지금까지 이 두 개를 헷갈려했거든요. "오늘 -1%인데 나는 +5%다" 이런 게 한눈에 보이면 안심이 돼요.

### 주식 고수 → UX
> 원화 환산은 필수인데, **실시간 환율**로 계산해야 합니다. 고정 환율이면 의미 없어요. 현재 USD/KRW API가 이미 있으니 연동하면 됩니다.

### 애널리스트 → PM
> "과거 종목" 매도 기록은 **세금 계산에도 필요**합니다. 해외 주식 양도세 250만원 공제 후 22% — 이걸 자동 계산해주면 차별화 포인트가 됩니다.

### FE → 전체
> 분할 매수 기록은 DB 스키마 변경이 필요합니다. 현재 `avgCost`, `shares`만 있는데, `transactions` 배열을 추가해야 합니다:
> ```
> transactions: [
>   { date: '2026-02-15', price: 430, shares: 10, type: 'buy' },
>   { date: '2026-03-05', price: 410, shares: 15, type: 'buy' },
> ]
> ```
> 이러면 평균 단가와 총 수량은 자동 계산됩니다.

## 최종 합의

### 카테고리 변경
```
Before: 단기 트레이딩 / 장기 투자 / 관심 종목
After:  투자 중 / 관심 종목 / 매도 완료
```

### 리스트 표시 (투자 중)
```
[아바타] 종목명(한글)  현재가  오늘등락  내수익/손실  목표달성도
```

### 종목 상세 카드 (탭 시)
```
현재가 vs 내 평균단가
오늘 등락 vs 내 수익/손실 (분리 표시)
보유수량 / 투자금 / 평가금 / 원화환산
목표 수익률 프로그레스 바
매수 이력 (날짜별)
```

### 즉시 추가 기능 5가지
1. 원화 환산
2. 분할 매수 기록
3. 카테고리 변경 (투자 중/관심/매도)
4. 내 수익/손실 리스트 표시
5. 목표 달성 프로그레스 바
