## 결정사항

### 1) 위계 반전 즉시 (P0) vs V1.2 분리?
**P0 즉시 채택**. 4분야 모두 동의 + 변경 폭 작음(3개 파일) + 베타 D-6 일정 영향 없음 + 시장 표준 일치는 베타 출시 전 필수. 사용자 첫 인상에 직접 영향.

### 2) 연관 종목 추천은 V1.2 후속
**왜**: ① 알고리즘 자체는 단순(`getRelatedSymbols`)이나 종목 상세 페이지 UI 변경 폭 큼 ② 법무 패널이 "베타 D-6엔 카피·약관·SSOT 검토 시간 부족, 출시 후 우선" ③ STOCK_KR 30종에 sector 필드 부재 — `koreanUniverse.ts` 신설 또는 STOCK_KR 확장 사전 작업 필요. **단 약관·FORBIDDEN_PHRASES 사전 준비는 P0에서 박제**.

### 3) "추천" 어휘 영구 금기
**왜**: 자본시장법 §6 투자자문업 시그널. FORBIDDEN_PHRASES에 이미 박제됨. 새 어휘로 **"같은 산업의 다른 종목"**(사실 진술, 자문 시그널 0)이 가장 안전. 약관 v3 제2조에 사전 박제.

### 4) PRO 차별점 (V1.2~V2)
**왜**: AI 촉 무료 원칙 유지 + PRO=비교 분석 카드는 자연스러운 페인포인트. 메모리 [project_monetization_model] PRO 도구만 잠금 원칙 일치.
